Playing with ng-Click

The ngClick directive allows you to specify custom behavior when an element is clicked.

Examples : Code

<div ng-init="count = 0">
	<button type="button" ng-click="count = count + 1">Increase count +</button>
	<button type="button" ng-click="count = count - 1">Decrease count -</button>
	<button type="button" ng-click="count = 0">Reset to zeor</button>
	<p><b>count : {{count}}</b></p>
</div>
