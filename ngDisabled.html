Playing with ng-Disabled

The ngDisabled directive sets the disabled attribute on the element (typically a form control, e.g. input, button, select etc.) if the expression inside ngDisabled evaluates to truthy.

Examples

<div>
	Check me to disable button : <input type="checkbox" ng-model="checked"/>
	<button ng-class="{'btn btn-primary': true, 'disabled': checked}" ng-disabled="checked">Click me</button>
</div>
