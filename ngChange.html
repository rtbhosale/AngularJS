Playing with ng-Change

The ngChange directive evaluates the given expression when the user changes the input. The expression is evaluated immediately, unlike the JavaScript onchange event which only triggers at the end of a change (usually, when the user leaves the form element or presses the return key).

ng-Change on checkbox, Examples : Code

<div ng-init="count = 0">
	<input id="angularjs" type="checkbox" ng-change="count = count + 1" ng-model="confirmed" />
	<p><b>count : {{count}}</b></p>
</div>

ng-Change on input, Examples : Code

<div ng-init="countText = 0">
	<label for="FirstName">Type in somthing to fire <span class="label label-primary">ngChange</span></label><br/>.
	<input id="FirstName" type="text" ng-change="countText = countText + 1" ng-model="firstName" />
	<p><b>Firstname : {{firstName}}</b></p>
	<p><b>count : {{countText}}</b></p>
</div>
